# 🎮 Tic-Tac-Toe Game (Python)

A simple and interactive 2-player Tic-Tac-Toe game built using Python. Players take turns placing X and O on a 3×3 grid. The game displays the board after each move, checks for winners, and handles invalid inputs gracefully. Perfect beginner project for logic building and game development.

---

## 📌 Features
- Clean and user-friendly board display  
- Input validation (no crashes)  
- Winner detection (rows, columns, diagonals)  
- Draw condition handling  
- Beginner-friendly code  

---

## 🧠 Game Rules
- Two players: X and O  
- Players enter row and column (0–2)  
- First to align 3 marks wins (horizontal, vertical, diagonal)  
- If all cells are filled → Draw

## 🚀 How to Run

1. Install Python  
2. Clone this repository  
   git clone https://github.com/BM804/tic_tac_Toe.git  

3. Go to project folder  
   cd tic-tac-toe-python  

4. Run the file  
   python tic_tac_toe.py
